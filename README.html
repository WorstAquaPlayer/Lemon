    Lemon

    Yarhl plugin for Nintendo 3DS common formats.

    The library supports .NET 5.0 and above on Linux, Window and MacOS.

    Release Package
    Stable
    Preview Azure Artifacts

    Supported formats

    Encryption, decryption or signature validation not supported yet.

    • NCSD (CCI and CSU): unpack
    • CIA: unpack and pack
    • NCCH (CXI and CFA): unpack
    • ExeFS: unpack and pack
    • RomFS: unpack and pack
    • TMD: deserialize

    Build

    The project requires to build .NET 5.0 SDK, .NET Core 3.1 runtime and .NET Framework 4.8 or latest Mono. If you open the project with VS Code and you did install the VS Code Remote Containers extension, you can have an already pre-configured development environment with Docker or Podman.

    To build, test and generate artifacts run:

    # Only required the first time
dotnet tool restore

# Default target is Stage-Artifacts
dotnet cake

    To just build and test quickly, run:

    dotnet cake --target=BuildTest
